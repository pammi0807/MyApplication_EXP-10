# MyApplication_EXP-10
Aim

To develop an Android application using RadioButton and RadioGroup to allow the user to select a gender option and display the selected choice.

Algorithm
1. Open Android Studio and create a new Android application project.
2. Design the user interface using LinearLayout in the XML file.
3. Add a TextView to display the heading “Select Gender”.
4. Add a RadioGroup to group multiple RadioButtons together.
5. Create RadioButtons for Male, Female, and Other options.
6. Add a Button to display the selected option.
7. Add another TextView to show the result.
8. Initialize all UI components in MainActivity using findViewById().
9. Set a click listener for the “Show Selected” button.
10. Use getCheckedRadioButtonId() method to identify the selected RadioButton.
11. Display a message if no option is selected; otherwise display the selected gender in the TextView.
12. Run the application and verify the RadioButton selection functionality successfully.
